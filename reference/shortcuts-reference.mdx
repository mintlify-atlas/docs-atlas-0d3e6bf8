Glyph provides keyboard shortcuts for common actions across the app and inside the editor. Application shortcuts fire globally ‚Äî even when focus is inside the editor.

<Note>
  On macOS, `‚åò` maps to the **Cmd** key. On Windows and Linux, `‚åò` maps to **Ctrl**. Glyph uses Electron's `CommandOrControl` accelerator internally, so both platforms are always covered.
</Note>

## Navigation

Use the navigation shortcuts to move through your note history without reaching for the mouse.

| Action | Description |
|--------|-------------|
| **Navigate Back** (`‚åò[` / `Ctrl+[`) | Go to the previous note in your navigation history. |
| **Navigate Forward** (`‚åò]` / `Ctrl+]`) | Go to the next note in your navigation history. |
