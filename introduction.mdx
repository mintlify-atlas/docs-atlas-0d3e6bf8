Glyph is a focused desktop workspace for people who want plain markdown files, beautiful reading, and a lightweight writing flow. You open local folders, move quickly between notes, and edit in a clean environment that stays out of your way.

**Current version: 0.3.0**

## Design philosophy

Glyph sits in the middle ground between plain-text tools and heavyweight note systems:

- **Plain files on disk.** Every note is a `.md`, `.mdx`, or `.markdown` file you own. There is no proprietary database or format to export from.
- **Beautiful reading.** The rendered view is typography-first — comfortable to read, not just functional.
- **Fast navigation.** The command palette, quick-open, and keyboard shortcuts let you reach any note without lifting your hands from the keyboard.
- **No lock-in.** Because your notes are plain files, you can open them in any editor, back them up with any tool, and move them anywhere at any time.

## Built with

Glyph is an [Electron](https://www.electronjs.org/) app with a React frontend. The key libraries powering the experience are:

- **[TipTap](https://tiptap.dev/)** — rich-text editor with full markdown support
- **[Zustand](https://zustand-demo.pmnd.rs/)** — lightweight state management
- **[Tailwind CSS v4](https://tailwindcss.com/)** — utility-first styling with OKLCH design tokens
