Type `/` at the start of any line in the editor to open the slash command menu. As you continue typing after `/`, the list filters to match your input ‚Äî select an item with the arrow keys and press **Enter**, or click it directly.

<Note>
  The slash command menu only triggers at the **start of a line**. If `/` is in the middle of existing text, it inserts a literal slash character.
</Note>

## Usage example

1. Place your cursor at the beginning of a blank line.
2. Type `/`.
3. Continue typing to filter ‚Äî for example, type `table` to jump straight to the **Table** command.
4. Press **Enter** (or click) to insert.

To dismiss the menu without inserting anything, press **Escape** or delete the `/` character.
