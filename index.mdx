title: "Welcome to Glyph"
description: "A local-first markdown editor and reader for macOS and Windows. Your notes stay as plain files — beautiful to read, fast to edit."
---

Glyph is a focused desktop workspace for people who want plain markdown files, beautiful reading, and a lightweight writing flow — without turning their notes into a database. Open local folders, navigate between notes instantly, and write in a clean editor that stays out of your way.

<Steps>
  <Step title="Install Glyph">
    Download the latest release for [macOS](/installation#macos) or [Windows](/installation#windows), or install via Homebrew with `brew install --cask FALAK097/glyph/glyph`.
  </Step>
  <Step title="Open a workspace">
    Launch Glyph and it will open your `Documents/Glyph` folder automatically. Use **Cmd/Ctrl+Shift+O** to open any local folder.
  </Step>
  <Step title="Start writing">
    Press **Cmd/Ctrl+N** to create a new note. Type `/` in the editor to insert tables, images, links, and more using slash commands.
  </Step>
  <Step title="Find anything fast">
    Press **Cmd/Ctrl+P** to open the command palette — search notes, switch files, and run commands from one place.
  </Step>
</Steps>
