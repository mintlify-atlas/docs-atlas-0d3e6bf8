This guide takes you from a fresh installation to writing your first note.

<Steps>
  <Step title="Launch Glyph">
    Open **Glyph** from Launchpad, Spotlight (macOS), or the Start menu (Windows).

    On first launch, Glyph automatically opens `Documents/Glyph` as your default workspace and creates the folder if it does not exist yet. The sidebar on the left shows your workspace tree.
  </Step>
  <Step title="Open a folder">
    To use a different folder as your workspace, press `Cmd+Shift+O` on macOS or `Ctrl+Shift+O` on Windows — or go to **File > Open Folder** in the menu bar.

    Select any local folder that contains markdown files. Glyph loads the entire folder tree into the sidebar, including nested subfolders.
  </Step>
  <Step title="Create a new note">
    Press `Cmd+N` (macOS) or `Ctrl+N` (Windows) to create a new note.

    Glyph creates a new file at the workspace root and opens it in the editor immediately. Rename it from the sidebar once you know what to call it.
  </Step>
  <Step title="Write with slash commands">
    Start typing in the editor. To insert rich content, type `/` at the **start of a line** to open the slash command menu.

    From the slash menu you can insert:

    - **Tables** — formatted markdown tables
    - **Links** — inline links with URL and label
    - **Images** — pick a local image from your machine

    Slash commands are the fastest way to add structure without leaving the keyboard.
  </Step>
  <Step title="Save your work">
    Glyph autosaves as you type — you do not need to save manually. If you want to force a save immediately, press `Cmd+S` (macOS) or `Ctrl+S` (Windows).

    Files are saved as plain markdown on disk, so they are always accessible from any other editor or file manager.
  </Step>
  <Step title="Navigate with the command palette">
    Press `Cmd+P` (macOS) or `Ctrl+P` (Windows) to open the command palette.

    From the command palette you can:

    - **Quick-open** any file in the workspace by name
    - **Search** your notes for a word or phrase
    - **Run commands** like toggling the sidebar, opening settings, or switching themes

    The command palette is the fastest way to move around without using the mouse.
  </Step>
</Steps>

<Tip>
  Press `Cmd+Shift+F` (macOS) or `Ctrl+Shift+F` (Windows) to toggle **focus mode**. Focus mode hides the sidebar and gives you a full-width editing canvas — useful when you want to write without distractions.
</Tip>

<Note>
  Glyph supports markdown link navigation. If a note contains a link to another `.md` file in the same workspace (for example `[My other note](other-note.md)`), clicking the link opens that file directly in the editor. Use `Cmd+[` / `Ctrl+[` and `Cmd+]` / `Ctrl+]` to navigate back and forward through your file history.
</Note>
